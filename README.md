# 高中物理教学演示

一个适合部署到 GitHub Pages 的静态网站，用来整理高中物理教学演示目录。

## 文件结构

- `index.html`: 页面结构
- `styles.css`: 页面样式
- `catalog-data.js`: 册次与章节目录
- `script.js`: 目录渲染、筛选与搜索
- `404.html`: 404 页面

## 当前目录组织

当前按 6 册结构组织：

- 必修第一册
- 必修第二册
- 必修第三册
- 选择性必修第一册
- 选择性必修第二册
- 选择性必修第三册

后续如果要补充具体教学演示，直接修改 `catalog-data.js` 即可。

## 后续怎么扩充

你可以按下面方式继续加内容：

1. 给某一章补充演示链接
2. 给某一章补充实验动画
3. 给某一章补充 PPT、讲义或题型入口
4. 把单章改成独立详情页

## 发布到 GitHub Pages

1. 在 GitHub 创建仓库。
2. 把当前目录文件上传到仓库根目录。
3. 进入仓库 `Settings` -> `Pages`。
4. 选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/ (root)` 目录。
6. 保存后等待 GitHub Pages 生成站点地址。
