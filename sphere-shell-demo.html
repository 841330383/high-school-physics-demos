接地球壳与球外点电荷

这个演示使用镜像电荷法构造接地导体球壳外部的精确静电场， 适合观察球面零电势、壳内场强为零、球外对称点电势关系以及外部电场线分布。

三维场景

蓝色为电场线，暖色半透明曲面为等势面，深色球面表示接地导体球壳。

电场线 等势面 接地球壳

边界条件

球面电势恒为 0，壳内场强恒为 0

求场方法

球外区域用“真实电荷 + 镜像电荷”精确重建

使用方式

三维看整体分布，二维专门拿来判定关键点

二维剖面

这里直接用于判定 A、B、O、M、N 的关系，并观察半径和外电荷位置变化。

A

球壳外的真实点电荷

B

壳内观察点，用来判断壳内合场

O

球心，用来区分“Q 单独作用”和“合场”

M

球外对称点，用于比较场强与电势

N

与 M 对称的球外观察点

易错点

壳内合场为 0 并不以接地为前提。本页采用接地情形，是为了把球面电势固定为 0， 并用最简的镜像电荷关系展示外场；同时不要把“Q 在 O 点单独产生的场”与“壳内实际合场”混为一谈。

模型介绍

接地导体球壳外有一个点电荷时，球外电场可以用“真实电荷 + 镜像电荷”来等效重建。 这个模型重点用于判断壳内是否为零场、镜像位置关系，以及球外对称点的电势和场强。
